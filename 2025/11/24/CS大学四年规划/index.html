985/211计算机新生：大学四年硬核学习路线规划

Created2025-11-24|Updated2025-11-24|CS修炼之路
|Post Views:

作为一名“92”院校（985/211）的计算机大一新生，目前的进度是 C 语言字符串。
深知计算机专业竞争激烈，目标不仅仅是“学会写代码”，而是成为一名底蕴深厚的工程师。

这是我为自己定制的大学四年硬核学习路线，立帖为证，死磕基础！👊

🚀 第一阶段：内功心法修炼（大一下 - 大二上）

关键词：指针、内存、数据结构、计算机系统

我现在学到字符串，其实正处于从“语法入门”跨向“内存理解”的关键节点。

  1. 攻克 C 语言 (地基)

    • 现状：字符串本质是字符数组，必须透彻理解内存地址。
    • 任务：
      • 死磕指针：搞懂指针与数组、函数、多级指针的关系。
      • 内存管理：理解 malloc/free，堆（Heap）与栈（Stack）的区别。
    • 书单：《C Primer Plus》、《C和指针》。

  2. 数据结构与算法 (内功)

    • 时机：C语言指针学完立刻开始。
    • 要求：手写代码！ 必须能用 C 语言手写链表反转、二叉树递归、快速排序。
    • 资源：《大话数据结构》、邓俊辉《数据结构》（B站）。

  3. 计算机系统导论 (神课)

    • 目标：大一读懂 CSAPP（深入理解计算机系统），打通软硬件任督二脉。

🛠️ 第二阶段：工业素养与工具链（大二）

关键词：OOP、Linux、网络、OS

C 语言造轮子，工程开发需要更高效的武器。

  1. 面向对象语言 (C++ / Java)

    • **首选 C++**：为了更好地理解底层和系统（配合 CSAPP）。
    • 备选 Java：为了日后走后端开发路线。
    • 重点：理解封装、继承、多态，以及内存模型。

  2. 操作系统 & 计算机网络 (面试双壁)

    • 操作系统：进程/线程、死锁、虚拟内存。推荐哈工大李治军老师网课。
    • 网络：TCP/IP 协议栈、三次握手、HTTP/HTTPS。

  3. 脱离学生气

    • Linux：抛弃 IDE，学会使用终端（Terminal）、Vim、GCC、Makefile。
    • Git：代码版本管理，开始维护自己的 GitHub 仓库。

⚔️ 第三阶段：实战与求职（大三）

关键词：LeetCode、造轮子、实习

  1. 算法刷题

    • 平台：LeetCode
    • 目标：300+ 道题（Medium 难度为主）。
    • 重点：动态规划、回溯、图论。

  2. 硬核项目 (拒绝管理系统)

    • 不要做“图书管理系统”。
    • 要做：
      • C++ 轻量级 Web 服务器（仿 TinyWebServer）。
      • 简单的操作系统内核（MIT 6.S081）。
      • RPC 框架实现。

  3. 大厂实习

    • 大三下学期（3-5月）投递暑期实习，这是进入大厂的捷径。

💡 给自己的几条铁律

  1. 数学不能丢：高数、线代、离散是 AI 和图形学的基石，高端代码全是数学。
  2. 英语是生产力：报错看 Stack Overflow，文档看官方英文版，别只看 CSDN。
  3. 重道轻术：不迷信框架（Spring Boot等），那是“术”；操作系统和网络原理才是“道”。作为 92 学生，根基必须扎实。

当前任务： 搞定 C 语言字符串 -> 拿下指针 -> 拿下结构体！
Keep Coding! 💻
